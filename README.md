📌 Project Overview
This is a front-end restaurant web application that dynamically loads menu items from a JSON data structure and renders them using JavaScript DOM manipulation.
The interface is styled using Tailwind CSS (CDN) to achieve a clean, responsive, and modern UI design.
The application includes product sliders, an interactive cart system, and smooth navigation between sections for improved user experience.

🛠️ Technologies Used
HTML5 – Semantic structure
Tailwind CSS (CDN) – Utility-first styling & responsiveness
JavaScript (Vanilla JS) – Dynamic rendering & functionality
JSON – Menu data storage and structured content handling

⚙️ Features Implemented
🎞️ Product Sliders
Horizontally scrollable food item sliders
Smooth transition effects
Dynamically populated from JSON data
🛒 Add to Cart Functionality
Add items to cart button
Cart updates dynamically using JavaScript
Quantity management logic
Real-time UI updates without page reload

🔄 Dynamic Data Rendering
Menu items stored in JSON format
JavaScript parses and renders items dynamically
Reusable card generation logic

🧭 Smooth Navigation
Smooth scrolling between sections
Navigation links connected to page anchors
Enhanced user experience

🧠 What This Project Demonstrates
DOM manipulation and dynamic UI rendering
Working with structured JSON data
Cart state management in front-end applications
Responsive design using utility-first CSS
Clean component-like code organization without frameworks

🚀 How to Run
Download or clone the repository
Open index.html in your browser
Ensure data.json is correctly linked
Interact with sliders and add items to the cart
No additional installation is required.
